# AfriPay Design System

Welcome to the **AfriPay Design System** - a comprehensive collection of reusable UI components built with **shadcn/ui**, **Radix UI**, and **Tailwind CSS**.

## 🎨 Design Principles

- **Accessibility First**: All components are built with accessibility in mind
- **Mobile Optimized**: Responsive design for mobile-first fintech experiences
- **African Context**: Color schemes and patterns tailored for African markets
- **Performance**: Lightweight and optimized components

## 🚀 Technology Stack

- **React 18** with TypeScript
- **shadcn/ui** component library
- **Radix UI** primitives for accessibility
- **Tailwind CSS** for styling
- **Lucide React** for icons

## 📱 AfriPay Features

This design system powers the AfriPay fintech application with:

- **Multi-role Support**: Consumer, Merchant, Agent, and Admin interfaces
- **Financial Services**: Wallets, transfers, bill payments, investments
- **E-commerce Integration**: Shopping, delivery, and marketplace features
- **KYC & Security**: Identity verification and security features
- **Multi-currency**: Support for various African currencies

## 🧩 Component Categories

### Core UI Components
- Buttons, Cards, Forms, Navigation
- Data Display: Tables, Charts, Progress indicators
- Feedback: Toasts, Alerts, Loading states

### Financial Components
- Wallet summaries, Transaction lists
- Payment forms, Currency displays
- Investment trackers, Savings goals

### Business Components
- Merchant dashboards, Order management
- Delivery tracking, Inventory management
- Agent tools, Admin panels
